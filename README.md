# mkproj

A lightweight Bash CLI tool for scaffolding Python projects inside WSL.

Designed for developers who work on Windows but code in VS Code using WSL.


## Features

- Creates project directory
- Optional git initialization
- Virtual environment setup
- README + requirements template
- Safe overwrite mode

## Usage

mkproj [options] <project_name>

Options:
  -f    Force overwrite
  -g    Initialize git
  -v    Show version
